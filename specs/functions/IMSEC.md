# IMSEC

## IMSEC

## Purpose
Returns secant of a complex number.

## Syntax
- Excel: `IMSEC(inumber)`
- Google Sheets: `IMSEC(inumber)`

## Behavior
- Computes secant in complex domain as reciprocal cosine.

## Examples (expected outputs)
- `IMSEC("1") -> 1.850815718`

## Error Cases
Invalid complex input returns error.

- Implemented in IronCalc.
