# IMCSC

## IMCSC

## Purpose
Returns cosecant of a complex number.

## Syntax
- Excel: `IMCSC(inumber)`
- Google Sheets: `IMCSC(inumber)`

## Behavior
- Computes reciprocal of sine in complex domain.

## Examples (expected outputs)
- `IMCSC("1") -> 1.188395105`

## Error Cases
Invalid complex input or undefined poles return errors.
