# N

## N

## Purpose
Converts input to numeric value.

## Syntax
- Excel: `N(value)`
- Google Sheets: `N(value)`

## Behavior
- Boolean/text conversion to number.

## Examples (expected outputs)
- `N(TRUE) -> 1`

## Error Cases
- Invalid argument count or input values return errors per host semantics.
