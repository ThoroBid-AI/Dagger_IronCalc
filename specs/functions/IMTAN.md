# IMTAN

## IMTAN

## Purpose
Returns tangent of a complex number.

## Syntax
- Excel: `IMTAN(inumber)`
- Google Sheets: `IMTAN(inumber)`

## Behavior
- Computes tan for complex input using sine/cosine ratio semantics.

## Examples (expected outputs)
- `IMTAN("1") -> 1.5574`

## Error Cases
- Invalid input returns error.
