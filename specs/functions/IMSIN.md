# IMSIN

## IMSIN

## Purpose
Returns sine of a complex number.

## Syntax
- Excel: `IMSIN(inumber)`
- Google Sheets: `IMSIN(inumber)`

## Behavior
- Computes sine for complex input with deterministic complex format.

## Examples (expected outputs)
- `IMSIN("1") -> 0.841470985`

## Error Cases
Invalid complex input returns error.
