# IMLOG10

## IMLOG10

## Purpose
Returns base-10 logarithm of a complex number.

## Syntax
- Excel: `IMLOG10(inumber)`
- Google Sheets: `IMLOG10(inumber)`

## Behavior
- Computes log base 10 on complex input.

## Examples (expected outputs)
- `IMLOG10("10") -> 1`

## Error Cases
Invalid complex input returns error.
