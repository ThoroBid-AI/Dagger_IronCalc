# ACOSH

## ACOSH


## Purpose
Return the inverse hyperbolic cosine of a number.

## Syntax
- Excel: `ACOSH(number)`
- Google Sheets: `ACOSH(value)`

## Behavior
- Input must be >= 1.

## Examples (expected outputs)
- `ACOSH(1)` -> `0`
- `ACOSH(10)` -> `2.9932228`

## Error Cases
- Input < 1 returns an error.

## Notes
- Implemented in IronCalc.

## Code Location
- Handler: `fn_acosh`
- File: `base/src/functions/mathematical.rs`
