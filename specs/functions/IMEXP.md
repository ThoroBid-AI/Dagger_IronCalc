# IMEXP

## IMEXP

## Purpose
Returns complex exponential.

## Syntax
- Excel: `IMEXP(inumber)`
- Google Sheets: `IMEXP(inumber)`

## Behavior
- Applies e^(z) in complex plane and returns normalized complex string.

## Examples (expected outputs)
- `IMEXP("1") -> 2.718281828`

## Error Cases
Invalid complex input returns error.
