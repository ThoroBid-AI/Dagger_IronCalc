# ISLOGICAL

## ISLOGICAL

## Purpose
Returns TRUE if value is logical TRUE/FALSE.

## Syntax
- Excel: `ISLOGICAL(value)`
- Google Sheets: `ISLOGICAL(value)`

## Behavior
- Type check for boolean values only.

## Examples (expected outputs)
- `ISLOGICAL(TRUE) -> TRUE`

## Error Cases
- Non-boolean values return FALSE.
