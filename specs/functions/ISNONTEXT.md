# ISNONTEXT

## ISNONTEXT

## Purpose
Returns TRUE if value is not text.

## Syntax
- Excel: `ISNONTEXT(value)`
- Google Sheets: `ISNONTEXT(value)`

## Behavior
- Performs strict type test for text values.

## Examples (expected outputs)
- `ISNONTEXT(5) -> TRUE`

## Error Cases
- No error conditions; returns FALSE for text.
