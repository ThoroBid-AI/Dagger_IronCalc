# IMLOG2

## IMLOG2

## Purpose
Returns base-2 logarithm of a complex number.

## Syntax
- Excel: `IMLOG2(inumber)`
- Google Sheets: `IMLOG2(inumber)`

## Behavior
- Computes log base 2 on complex input.

## Examples (expected outputs)
- `IMLOG2("4") -> 2`

## Error Cases
Invalid complex input returns error.
