# Function Specs

This directory is reserved for per-function specification files.

Current state:
- The active Lua planning workflow can run from `specs/lua_prompts.md`.
- Add `specs/functions/<FUNCTION>.md` files over time for deeper canonical specs.

When function spec files exist, they should include at least:
- Purpose
- Syntax
- Behavior
- Examples
- Error Cases
- Notes
- Code Location
